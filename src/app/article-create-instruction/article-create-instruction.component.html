Instruction

Article consists of

  • Article object with primary language and optional source title, url and author name.
  • Translations array with title, language and optional short description and image url.
  • Text versions array with article text.

So, Article object can looks like this:

{{exampleArticleObject | json}}

Everyone can create new article, translation or text version.

Article can be approved or published only by admins.

To become published, translation or text version must be approved by admin or author of parent object.

Also, author of parent object can approve, publish, edit and delete translations and text version.

You can edit only single object at once. I.e. article or translation or text version.

You can select which translation or text version edit on article page.

You can't delete the only text version of translation or the only translation in article.
